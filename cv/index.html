Projects

  • 2023.09 - Present
    Neural Activity Information Decoding Based on Brain Imaging (CAS Strategic Priority Program)
    Developed decoding models for visual perception based on high-dimensional spatiotemporal EEG signals within a national strategic priority program.
    • Designed a spatiotemporal feature fusion decoding network tailored for EEG signals.
    • Built a dual-stream cross-modal decoding architecture for brain-visual alignment.
    • Used deep residual networks to extract high-level semantic features from visual images.
    • Applied multi-head self-attention to capture millisecond-level temporal dynamics of brain signals.
    • Achieved high-precision semantic alignment in latent space via contrastive learning.
  • 2024.01 - Present
    Neuroscience-Inspired Brain-Vision-Language Tri-Modality Alignment Framework (Guangdong Provincial Key Laboratory Project)
    Constructed a large model-driven tri-modality semantic fusion framework for brain, vision, and language in non-invasive BCI.
    • Built a joint embedding space for brain, vision, and language modalities.
    • Introduced large language models to generate fine-grained text descriptions as semantic anchors.
    • Enforced high-precision alignment of brain signals with visual and linguistic features via cross-modal contrastive learning.
    • Proposed a dynamic loss adjustment strategy to balance low-level features and high-level semantics.
    • Simulated perception-to-cognition processes inspired by the dual-stream visual system.
  • 2022.01 - 2023.05
    FPGA Code Rule Checker and Circuit Display Development (National Outsourcing Project)
    Led an undergraduate sub-project on FPGA code rule checking and circuit visualization for VHDL and Verilog.
    • Designed the overall scheme and managed software development and project progress.
    • Implemented a code rule checker for VHDL and Verilog.
    • Developed DAG-based topological sorting and automatic layout algorithms.
    • Realized real-time rendering and visualization of logic circuit diagrams corresponding to violating code segments.
    • Addressed the traceability problem from code to circuit in complex logic designs.
